The Project

JustCook was built to solve a real household problem — to be specific, my own problem. I approach development by starting with the problem, not the technology. For this project, I analyzed existing tools and identified a gap in how real households plan meals, which led to a rotation-first design. I built a lightweight recommendation engine using rule-based scoring to keep the system predictable and efficient, avoiding unnecessary complexity. On the frontend, I used RxJS for state management to keep data flow simple without introducing heavy frameworks. On the backend, I designed a relational model and scalable APIs with .NET and EF Core, including aggregation logic for features like grocery list generation. The system is structured using clean architecture principles, with cloud storage integrated for handling media efficiently without impacting database performance.

Full-stack developer focused on building practical, problem-driven applications with clean architecture and simple, effective solutions. After a career break in 2023, I built JustCook to demonstrate that my skills remain current — and to show I think about why I'm building something, not just how.
